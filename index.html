Manual da linguagem de marcação: HTML 5


Está página tem objetivo de apresentar as principais TAGS do HTML utilizados para desenvolvimento WEB. Com isso, ajudará o programador iniciante a desenvolver o programa eficaz e da maneira correta.

As principais TAGS utilizadas são:

  • Tittle - Serve para nomear as abas da páginas.
  • h1 - Utilizado para definir que o contéudo é um título.
  • p - Utilizado para definir que é um parágrafo.
  • img - Utilizado para exibir uma imagem em uma página HTML.
  • br - Utilizado para quebrar linha.
  • ul - Exibir uma lista não ordenada.
  • ol - Exibir uma lista ordenada.
  • dl - Exibir uma lista com descrição.
  • a - Define um HiperLink.
